Strativ AB Technical Screening Test
As part of the screening test, you have to make two simple GET and POST requests to two endpoints.

* Be sure to pass your valid email as part of the query parameter: email for both requests.

The POST Request

  • POST: https://screeningtest-1-q9276056.deta.app/token/
  • Header: X-API-KEY = The last name of the father of analysis algorithms. (For example: John Doe, the API key will be "doe")
  • After a successful POST request, you will receive a JWT token. Use that token as the token header for the GET request.

The GET Request

  • GET: https://screeningtest-1-q9276056.deta.app/url/
  • Header: token = The token you received from the previous step
  • This final API will return url to proceed to the next step.

Good luck!
