Strativ AB Techincal Screening Test

You have a to make simple GET and POST requests to two endpoints using an API key.

The API key is the last name of the person who created the C# language. (Ex: John Doe, API-Key=doe)

Be sure to pass your valid email as part of the query parameter: email.

After a successful POST request, you will receive a JWT token. Use that token as the token header for the GET request.

This final API will return an encoded string. Decode the string to proceed to the next step.

Notable URLs:

  • https://screeningtest-1-q9276056.deta.app/token/ : POST endpoint to generate JWT token

  • https://screeningtest-1-q9276056.deta.app/url/" : GET endpoint for an encoded string



* HINT 1: The POST API call should contain your API key in the X-API-KEY header

* HINT 2: All API requests should have the Content-Type: 'application.json' header and the token
